### 問題
1.都是子目錄為什麼有時候是一般folder有時候又稱為repo
hint:
- 隱藏檔案
- 在一般的子目錄下GIT指令會怎樣

### 練習
1. 多人專案，發生衝突的解決
1. 複製別人的repo 練習殼層指令
1. 利用AI 

- `cd` 後面接上路徑名稱 如果為空白 則是 `家目錄`
- `pwd` 
- `1s` 列出目前目錄的檔案
- `1s -a`會顯示隱藏檔案 但只有檔案名稱
- `1s -1a` 
- `～` 是甚麼意思 代表家目錄
- `cp`
- `cp -r 資料夾/* 新資料夾` 
-  `cp 檔案 新檔案名稱`  複製
-  `mkdir 新資料夾名稱`  建立
-  `rm` 刪除檔案
